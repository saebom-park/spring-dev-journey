# 🧷 02. Spring Core

> 이 단계는 스프링 프레임워크의 핵심 개념을 학습합니다.  
> IoC, DI, Bean 등록 등 스프링의 철학과 동작 방식을 이해하고, 실무 개발의 토대를 다집니다.

---

## ✅ 학습 목표

- 스프링 컨테이너의 동작 방식 이해
- 의존성 주입(DI)의 개념과 실습 적용
- 다양한 Bean 등록 방식과 생명주기 학습
- 설정 클래스 기반의 Java Config 스타일에 익숙해지기

---

## 📚 학습 주제

| 주제 | 설명 |
|------|------|
| IoC (제어의 역전) | 객체 생성 및 생명주기 관리 책임을 스프링이 담당 |
| DI (의존성 주입) | 필요한 객체를 외부에서 주입받는 방식 |
| Bean 등록 | `@Component`, `@Bean`, 수동 등록 등 다양한 방법 |
| 스프링 컨테이너 | ApplicationContext의 역할과 특징 |
| 설정 클래스 | `@Configuration`, `@ComponentScan`, `@Import` |
| 의존 관계 설정 | 생성자 주입 vs 필드 주입 vs Setter 주입 |
| 싱글톤, 프로토타입 | 스코프(scope)에 따른 빈의 생성 전략 |
