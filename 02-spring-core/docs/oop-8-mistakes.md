### 😅 실수 1 — 커피머신 호출이 main에 남아있었음

```java
a.orderCoffee(b);
c.brew(b)
```

✅ 정답:

```java
a.orderCoffee(b);
// Barista 내부에서 coffeeMachine.brew() 호출
```

📌 **설명**:

- 커피를 준비하는 책임은 **Barista에게 있음**
- `brew()` 호출이 main에 남아있으면 **협력 흐름이 외부로 끊어짐**
- 메시지 흐름이 자연스러우려면 **"시킨 사람 → 처리한 사람 → 자원"** 순서가 내부에 포함돼야 함

---

### 📌 요약 포인트

- 커피머신은 **Barista가 직접 호출**해야 함
- 외부에서 연결하면 협력 구조가 끊어짐
- 메시지 흐름은 **내부에서 끝까지 이어져야 자연스러움**
