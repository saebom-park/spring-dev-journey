# [OOP-4단계] 질문노트: 오버라이딩 개념 (SenderExample.java)

> 💬 봄이가 실습 도중 궁금했던 내용들 정리
> 

| 질문 | 답변 요약 |
| --- | --- |
| `super`는 부모 메서드인데 왜 `a2.super.speak()`는 안 돼? | `super`는 **객체처럼 참조할 수 있는 게 아니라**, **자식 클래스 내부에서만 쓸 수 있는 예약어**라서 그래. `a2`는 단순 객체 참조니까 `super`랑 붙일 수 없어. 대신 `super.speak()`는 자식 클래스 안에서만 쓸 수 있어! |
| 오버라이딩된 메서드는 무조건 자식 메서드가 먼저 실행돼? | 맞아! 오버라이딩된 메서드는 실행 시 **자식 클래스 버전이 우선 실행**돼. 자바는 객체가 어떤 타입인지보다 실제 생성된 인스턴스를 기준으로 메서드를 찾아서 실행해. |
| `@Override` 꼭 붙여야 돼? | 문법적으로는 생략해도 돼. 하지만 **오타나 시그니처 불일치가 있을 때 컴파일 에러로 잡아줘서** 실무에서는 거의 필수처럼 사용돼. 가독성도 훨씬 좋아지고! |
