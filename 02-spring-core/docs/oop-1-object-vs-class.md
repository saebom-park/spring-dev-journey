# [OOP-1단계] 객체 vs 클래스 차이

> “객체랑 클래스는 도대체 뭐가 어떻게 다른데?”
> 
> 
> 👉 **클래스는 설계도, 객체는 실체!**
> 

---

### 💡 핵심 개념 요약

| 개념 | 설명 |
| --- | --- |
| 클래스 (class) | 객체를 만들기 위한 설계도 |
| 객체 (object) | 클래스를 바탕으로 만들어진 실체 |
| 인스턴스화 | `new 클래스명()`으로 객체를 생성하는 행위 |
| 참조 변수 | 객체를 가리키는 변수, 객체의 주소를 저장함 |

---

### 🧾 예시 코드

```java
// 클래스 정의
public class Car {
    String model;
    int speed;

    void drive() {
        System.out.println("부릉부릉~ 달립니다!");
    }
}

// 객체 생성 & 사용
public class Main {
    public static void main(String[] args) {
        Car myCar = new Car();         // 객체 생성
        myCar.model = "소나타";         // 속성 설정
        myCar.speed = 100;
        myCar.drive();                 // 메서드 호출
    }
}
```

---

### 📌 포인트 요약

- 클래스: 설계도, 틀 (속성과 기능 정의)
- 객체: 설계도로 찍어낸 실체 (메모리에 올라가는 실제 데이터)
- `new 클래스()`로 객체 생성
- 생성된 객체는 **참조 변수**를 통해 접근
