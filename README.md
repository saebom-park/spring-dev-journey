## 🧾 학습 원칙

- 자동완성 최소화 → **손코딩 중심 학습**
- 모든 실습은 **Controller → Service → Repository** 3계층 구조 준수
- JPA 실습은 **Entity ↔ Repository ↔ Service ↔ DTO ↔ Controller** 를 명확히 분리
- 모든 코드/실습은 `.md` 문서와 함께 기록
- 질문/실수/개념 정리는 자동으로 분리하여 문서화  
  (`-questions.md`, `-mistakes.md`, `-details.md`)
