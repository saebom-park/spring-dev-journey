# 🌿 spring-dev-journey

> Java → Spring → DB → API → 보안 → 배포 → 실전 서비스까지  
> 백엔드 실무 역량을 체계적으로 구축하기 위한 Spring 기반 학습 프로젝트입니다.  
> 모든 학습은 손코딩 중심, 실습 중심, 문서화 중심으로 진행됩니다.

## 🚀 전체 학습 흐름 요약

1. **Java 문법 + 객체지향 기초 다지기**
2. **Spring Core 기반의 DI/IoC, 계층 구조, REST API 기본기 이해**
3. **JPA/Hibernate로 DB 연동 구조 심화**
4. **실전 API 설계 & 검증/예외 처리 학습**
5. **Spring Security로 인증/인가 체계 구축**
6. **Security All-in-One으로 세션 → JWT → 필터체인 → 인가 전체 흐름 통합**
7. **EC2 + Nginx + HTTPS + CI/CD로 배포 환경 구축**
8. **도메인 기반 실무 프로젝트로 서비스 설계 및 구현 역량 강화**
9. **Review Sprint에서 JDBC·MyBatis·JPA CRUD를 반복 학습하여 체화**
10. **Reference 섹션에서 CS 핵심 개념 보완**

---

## 🧾 학습 원칙

- 자동완성 기능 OFF → **모든 코드를 직접 손으로 작성하며 학습**
- 실습은 항상 **Controller → Service → Repository** 구조 준수
- JPA 실습은 **Entity ↔ Repository ↔ Service ↔ DTO ↔ Controller** 를 명확히 분리
- 모든 실습은 `.md` 문서로 기록 (예시 코드 + 흐름 설명 포함)
- 질문/실수/추가 개념 정리는 자동 분리  
  (`-questions.md`, `-mistakes.md`, `-details.md`)
