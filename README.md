# Spring Dev Journey

이 저장소는 자바 백엔드 실무 개발자가 되기 위한 학습 기록입니다.  
기초 문법부터 실무 적용까지의 과정을 체계적으로 정리하며, 매일 실습하고 기록합니다.

---

## 📁 폴더 구조

```
spring-dev-journey/
├── 01-core-java/
│   ├── 01-class-object.md
│   ├── 02-constructor.md
│   ├── 03-method-overloading.md
│   ├── 04-method-return-if.md
│   ├── 05-encapsulation.md
│   └── ...
```

---

## 학습 흐름

1. 자바 기초 문법 (클래스, 생성자, 캡슐화, 상속 등)
2. 객체지향 개념 적용
3. 예외 처리, 컬렉션, 반복문 등 확장 개념
4. 실전 API 개발과 스프링 연동 예정

---

## 학습 기록 방식

- 각 `.md` 파일은 단계별 수업 내용을 정리한 문서입니다.
- 예시 코드, 핵심 개념 요약, 실습 미션으로 구성되어 있습니다.
- 매일의 실수나 깨달음은 따로 기록합니다.
