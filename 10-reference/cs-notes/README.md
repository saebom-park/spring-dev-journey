# 10. CS 정리 노트 (cs-notes)
백엔드 엔지니어로 성장하기 위해 필수적인 **CS 전 영역을 체계화하여 정리한 문서 저장소**입니다.  
운영체제, 네트워크, 데이터베이스, 디자인 패턴 등 핵심 기초지식을  
**Day 단위로 구조화**해 개념을 정확하게 이해하고 실무에 적용할 수 있도록 구성했습니다.

---

# 🎯 프로젝트 목표

- 백엔드 개발에 필요한 **핵심 CS 지식의 체계적 정리**  
- 개념 · 구조 · 동작 원리에 대한 **정확한 기술적 이해**  
- 실제 시스템 구현 또는 트러블슈팅에 연결되는 **실무 관점 기반의 설명**  
- 장기적으로 참고 가능한 **지속 가능한 CS 아카이브 구축**

---

# 📂 폴더 구조

```
/cs
 ├── os/               # 운영체제 지식
 ├── network/          # 네트워크 프로토콜·통신 구조
 ├── database/         # 데이터베이스 내부 동작 및 트랜잭션 구조
 ├── dev-basic/        # 개발자 필수 기본기
 ├── design-pattern/   # 객체지향 설계 및 GoF 디자인패턴
 ├── docs/       # 전체 커리큘럼 (v1, v2) 및 문서 규칙
 └── README.md
```

각 주제 폴더 안에는 Day 단위 개념 문서가 포함됩니다.

# 🌱 학습 방식

1. Day 단위로 핵심 개념을 정리  
2. 개념을 실제 기술 언어로 설명하는 연습  
3. 유관 개념 간 비교와 연결 고리 파악  
4. 실무 환경에서 관련 이슈가 발생했을 때  
   어떤 원리로 접근해야 하는지에 대한 사고 방식 정립
