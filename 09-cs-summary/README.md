# 🧷 09. CS 정리 노트 (CS Summary)

> 이 폴더는 자바 백엔드 개발자로서 꼭 알고 있어야 할  
> **컴퓨터 공학 핵심 개념(CS)**들을 정리해두는 공간입니다.  
> 네트워크, 운영체제, DB, 자료구조, 스레드 등 실무와 면접 모두에 필요한 개념을 중심으로 구성합니다.

---

## 📌 정리 목적

- 실무와 면접에 필요한 CS 지식을 지속적으로 누적
- 외우기보다 **이해 중심**으로 흐름과 구조를 정리
- 구글링 → 요약 → 내 언어로 정리하는 방식 추구

---

## 🧠 주요 정리 영역

| 카테고리 | 예시 키워드 |
|----------|--------------|
| 네트워크 | TCP/UDP, 3-way-handshake, HTTP/1~3, 쿠키/세션, CORS |
| 운영체제 | 프로세스/스레드, 컨텍스트 스위칭, 메모리 구조, 스케줄링 |
| 자료구조 | 배열, 연결 리스트, 스택/큐, 해시맵, 트리, 힙 |
| 알고리즘 | 정렬, 탐색, 시간복잡도, 그리디, DFS/BFS |
| 데이터베이스 | 인덱스, 조인, 정규화, 트랜잭션, 락, 격리 수준 |
| 자바 & JVM | 힙/스택, GC, 클래스 로딩, 동기화, 예외 처리 |
| 기타 | Git 전략, HTTP 상태코드, REST 개념, 인코딩 등 |

## ✨ 정리 방식 (임시 기준)

- 처음에는 주제별 `.md` 파일 단위로 작성
- 향후 분량 늘어나면 디렉토리별 분리 (`network/`, `os/` 등)
- 키워드 → 정의 → 예시 → 실제 적용 흐름으로 정리 권장
