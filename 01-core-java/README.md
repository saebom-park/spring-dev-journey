# 01-core-java
Java 기본 문법을 익히고 객체지향으로 확장하기 위한 기초를 다지는 단계입니다.  
클래스, 객체, 조건문, 배열, 상속 등 핵심 개념을 손코딩을 통해 탄탄하게 구축합니다.

---

## 🗂️ 구성 안내
- `docs/` : 각 단계별 개념 요약, 예시 코드, 실습 미션
- `src/`  : 손코딩 실습 코드 모음

---

## 🎯 학습 목표
- Java 기본 문법을 이해하고 직접 구현할 수 있다.
- 객체지향의 4대 원칙(캡슐화, 상속, 추상화, 다형성)을 코드로 표현할 수 있다.
- 이후 Spring의 DI/IoC, 도메인 모델링을 이해하는 기반을 마련한다.
- 실습 중 발생한 오류·실수는 문서화하여 반복 학습 자산으로 만든다.

---

## 🧠 전체 커리큘럼 (총 16단계)

| 단계 | 주제           | 주요 개념                                |
|----|--------------|--------------------------------------|
| 01 | 클래스 & 객체     | `class`, `new`, `static`, `void`     |
| 02 | 생성자 & this   | 생성자, 매개변수, `this`                    |
| 03 | 메서드 오버로딩     | 중복 정의, 시그니처                          |
| 04 | return & 조건문 | `return`, `if`, `else`               |
| 05 | 접근 제어자 & 캡슐화 | `private`, getter/setter             |
| 06 | 배열           | 선언, 초기화, index                       |
| 07 | 컬렉션          | `ArrayList`, `HashMap`, `HashSet`    |
| 08 | 반복문          | `for`, `while`, `do-while`, for-each |
| 09 | 포함 관계(has-a) | 필드 포함, 객체 협력                         |
| 10 | 상속           | `extends`, `@Override`, `super`      |
| 11 | 추상 클래스       | abstract class, 추상 메서드               |
| 12 | 인터페이스        | interface, 다형성                       |
| 13 | 예외 처리        | try/catch, throw, Custom Exception   |
| 14 | static 심화    | 정적 메서드/변수                            |
| 15 | 날짜/시간 API    | `LocalDate`, `DateTimeFormatter`     |
| 16 | Java I/O     | `Scanner`, `BufferedReader`, `File`  |

## 📂 파일 구성

각 단계는 다음 규칙을 기준으로 문서를 구성합니다.

| 파일명 형식                  | 설명                               |
|-------------------------|----------------------------------|
| `NN-topic.md`           | 기본 수업자료 (개념 요약, 예시 코드, 실습 미션 포함) |
| `NN-topic-questions.md` | 학습 중 나온 질문과 답변 정리 (필요 시 생성)      |
| `NN-topic-mistakes.md`  | 실습 중 발생한 오류·실수 기록 (필요 시 생성)      |
| `NN-topic-extra.md`     | 심화 개념 및 추가 설명 정리 (필요 시 생성)       |

> 단계에 따라 `-questions`, `-mistakes`, `-extra` 파일은 생략될 수 있습니다.

---

## 🧭 학습 흐름

1. 각 단계의 `.md` 문서에서 핵심 개념을 먼저 정리합니다.
2. 제공된 예시 코드를 이해한 뒤 직접 손코딩 실습을 진행합니다.
3. 오류 또는 실수는 `-mistakes.md`에 기록하여 반복 학습에 활용합니다.
4. 추가로 이해가 필요한 개념은 `-extra.md`에서 심화 학습합니다.
5. 학습 중 떠오른 질문은 `-questions.md`에 Q/A 형식으로 정리합니다.

---

## 📌 문서 종류별 기록 원칙

### **`-extra.md`**
- 핵심 개념을 심화 정리하는 문서
- 표, 예시 코드, 요약을 포함해 구조적으로 정리
- 실무 기준 설명을 간결하게 포함

### **`-questions.md`**
- 학습 중 나온 질문을 Q/A 형식으로 정리
- 질문은 그대로 기록하고 답변은 핵심만 요약
- 실무 적용 관점을 우선적으로 설명

### **`-mistakes.md`**
- 실습 중 발생한 실수를 실제 코드와 함께 기록
- 잘못된 코드 → 수정 코드 → 원인 → 요약 순으로 정리
- 반복 실수를 방지할 수 있도록 오류 패턴 중심으로 구성
