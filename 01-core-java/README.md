# 01. Core Java

자바 기초 문법 학습 단계입니다.  
클래스, 객체, 조건문, 배열, 상속 등 핵심 개념을 실습하며 이해합니다.

📁 수업 자료는 `docs/` 폴더에 정리되어 있습니다.  
📁 실습 코드는 `src/` 폴더에 작성되어 있습니다.

---

## ✅ 학습 목표

- 자바 문법을 명확히 이해하고 자유롭게 실습할 수 있다.
- 객체지향의 4대 원칙 (캡슐화, 상속, 추상화, 다형성)을 코드로 표현할 수 있다.
- 실수를 반복 학습 자산으로 전환하는 습관을 만든다.

---

## 🧠 전체 학습 구성 (총 16단계)

| 단계 | 주제 | 주요 개념 키워드 |
|------|------|------------------|
| 01단계 | 클래스 & 객체 | `class`, `new`, `static`, `void` |
| 02단계 | 생성자 & this | 생성자, 매개변수, `this` |
| 03단계 | 메서드 오버로딩 | 메서드 중복 정의, 시그니처 |
| 04단계 | return & 조건문 | `return`, `if`, `else` |
| 05단계 | 접근 제어자 & 캡슐화 | `private`, `getter/setter`, 정보 은닉 |
| 06단계 | 배열 | 배열 선언, 초기화, 인덱스 접근 |
| 07단계 | 컬렉션 | `ArrayList`, `HashMap`, `HashSet` |
| 08단계 | 반복문 | `for`, `while`, `do-while`, for-each |
| 09단계 | 포함 관계 (has-a) | 객체 포함, 필드 객체화, `address.show()` |
| 10단계 | 상속 | `extends`, `@Override`, `super()` |
| 11단계 | 추상 클래스 | `abstract class`, 추상 메서드, 강제 구현 |
| 12단계 | 인터페이스 | `interface`, `implements`, 다형성 |
| 13단계 | 예외 처리 | `try`, `catch`, `throw`, 사용자 예외 |
| 14단계 | static 심화 | 정적 변수/메서드, 팩토리 메서드 |
| 15단계 | 날짜 & 시간 API | `LocalDate`, `DateTimeFormatter` |
| 16단계 | 자바 I/O | `Scanner`, `BufferedReader`, `File` |

---

## 🧭 학습 흐름

1. 각 단계의 `단계명.md` 파일을 차례대로 학습
2. 예시 코드 → 직접 실습
3. 실습 중 발생한 오류는 `-mistakes.md`로 복습
4. 이해가 더 필요한 개념은 `-extra.md`로 확장 학습

## 📌 정리 원칙

- 개념 요약은 반드시 표로 명확하게
- 모든 개념에는 최소 1개 이상의 예시 코드 포함
- 마크다운 문서 내에는 항상 실습 미션이 포함됨
- 실수 노트는 붉은 느낌 이모지와 함께 정리하여 시각적으로 강조
- 추가 개념은 질문 중심 → 실무 연계 가능성 고려
