### 💡 1. `super("문자열")`은 왜 쓰는 건가요?

| 질문 | 답변 요약 |
| --- | --- |
| `super("문자열")`은 왜 써요? | 부모 클래스인 `Exception`의 생성자에 메시지를 전달해서, 나중에 `getMessage()`로 꺼내쓸 수 있도록 하기 위해서예요. |
| 그럼 일반 클래스에서 `super(name, age)`랑 똑같은 건가요? | 기본적으로 부모 생성자를 호출한다는 점은 같지만, 일반 클래스는 여러 필드를 초기화하고, 예외 클래스는 주로 message 한 가지만 전달해요. |
| super 안에 뭐 넣느냐에 따라 호출되는 생성자가 달라요? | 맞아요! 전달된 값의 타입과 개수에 따라 자동으로 매칭되는 부모 생성자가 호출돼요. |

---

### 🌱 정리 키워드

- `super(...)`는 부모 생성자 호출 키워드
- 예외 클래스는 `Exception(String)` 생성자를 주로 사용
- 일반 클래스보다 훨씬 단순한 구조 (message 1개 전달)
- 오버로딩된 생성자 중 타입/개수 맞는 걸 자바가 자동으로 호출함
