# Day01 | 짝수와 홀수 (프로그래머스)

🔗 문제 바로가기

---

### 💡 내가 푼 방식 (봄이)

```java
public String solution(int num) {
    if (num % 2 == 0) {
        return "Even";
    } else {
        return "Odd";
    }
}
```

- if-else 구조로 명확하게 분기
- 조건식에 대한 이해력 굿
- 직관적이고 안정적인 풀이 스타일

---

### 💡 온이 제안 방식 (삼항 연산자 리팩토링)

```java
public String solution(int num) {
    return (num % 2 == 0) ? "Even" : "Odd";
}
```

- 삼항 연산자로 분기 간결화
- 실무에서도 자주 사용하는 패턴
- 간단한 조건일 땐 훨씬 짧고 가독성 좋음

---

### 📌 비교 포인트 요약

| 항목 | 봄이 풀이 | 온이 풀이 |
| --- | --- | --- |
| 코드 구조 | if ~ else | 삼항 연산자 |
| 길이 | 약간 길지만 명확 | 한 줄 처리로 간결 |
| 가독성 | 누구나 읽기 쉬움 | 조건이 짧을 때만 추천 |
| 실무 활용성 | 범용적 | 실무 단순 분기에서 자주 사용됨 |

---

### 🌱 핵심 개념 요약

- **조건문**
    - if, else, 삼항 연산자 `? :`
- **짝수 판별**
    - `%` 연산으로 판별 → `num % 2 == 0`이면 짝수
