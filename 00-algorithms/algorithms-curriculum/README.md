# ☀️ Algorithm Curriculum

> 📍 위치: `00-algorithms/algorithms-curriculum/`
>
> 인강 없이, 교재형 포맷으로 진행하는 3개월 알고리즘 루틴.  
> 목표는 **“문제를 설계하고 설명할 수 있는 수준의 실전 감각 확보.”**

---

## 📘 전체 Phase 구성

| Phase | 주제 | 핵심 목표 | 진행상태 |
|--------|------|-----------|-----------|
| **Phase 0** | Java 문법 리빌드 | 손코딩 기반 문법 체화 | ✅ 완료 |
| **Phase 1** | 구현 / 수학 / 문자열 | 기본 알고리즘 감각 확립 | 🔄 진행 중 |
| **Phase 2** | 자료구조 / 정렬 / 그리디 | 효율적 사고력 구축 | ⏳ 예정 |
| **Phase 3** | 탐색 (DFS/BFS/백트래킹) | 구조적 탐색 패턴 완성 | ⏳ 예정 |
| **Phase 4** | DP / 최적화 / 트리 / 그래프 | 문제 설계 및 최적화 사고 완성 | ⏳ 예정 |
| **Phase 5** | 실전 모의 테스트 | 3문제/90분 완주 | ⏳ 예정 |

---

## 🧭 진행 방식

1. **Phase 0** — `day-file-skeleton.md` 기반, 문법 손코딩 완성  
2. **Phase 1~5** — `concept-v3.md` + `day-file-skeleton.md` + `weekly-review.md`  
3. **매일 루틴:**  
   - 💡 개념 요약 → 🧾 문제 풀이 → 📌 비교 포인트 → 🌱 핵심 정리  
   - 📒 질문 / 🐞 실수 / 🔁 리마인드 섹션 필수 작성  
4. **매주 일요일:** `weekly-review.md` 로 회고 및 복습
