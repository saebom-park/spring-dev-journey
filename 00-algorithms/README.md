# 00. Algorithms

이 폴더는 spring-dev-journey 프로젝트에서  
**Day 단위로 알고리즘 문제를 풀고 정리하는 루틴**을 담고 있습니다.

---

## ✅ 운영 방식

- 하루 1문제씩 진행하는 **Day 루틴 방식**  
- Day 단위로 `.md` 파일에 정리  
- 7문제 단위로 `routine01/`, `routine02/` 폴더로 구분
- 문제는 프로그래머스, 백준 혼합
- 개념 학습보다는 **실전 문제풀이 속 개념 습득**에 집중

---

## 🧾 정리 포맷

각 문제는 아래 구조로 정리됩니다:

```
💡 개념 요약
🧾 예시 코드 (내 풀이 vs 온이 풀이)
📌 비교 포인트
🌱 핵심 개념
```

- 실수했을 경우 → `dayXX-mistakes.md`
- 질문이 생겼을 경우 → `dayXX-questions.md`

---

## 📂 폴더 구조 예시

```
00-algorithms/
├── routine01/
│   ├── day01-even-or-odd.md
│   ├── day02-average.md
│   └── ...
├── routine02/
│   └── day08-xxx.md
└── README.md
```

---

## 🎯 학습 목표

- 대기업 코딩테스트 필수 개념 마스터
- 실전 기반 문제풀이 경험 축적
- 반복과 복기를 통한 사고력 향상
- 나만의 `.md` 기록 기반 알고리즘 포트폴리오 구축
