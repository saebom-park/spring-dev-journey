# 📋 회원제 게시판 시스템 (bulletin-board)

> 로그인한 회원만 글을 쓰고 댓글을 달 수 있는  
> **기본에 충실한 CRUD 게시판 서비스**입니다.  
> 실무 흐름을 기반으로 프론트(Vue) + 백엔드(Spring Boot)를 분리 개발합니다.

---

## 💡 시나리오

> 회원은 회원가입 및 로그인 후 게시글을 작성할 수 있으며,  
> 다른 회원의 글을 열람하고 댓글을 남길 수 있습니다.  
> 본인이 작성한 글과 댓글만 수정/삭제할 수 있습니다.

---

## 📋 요구사항

### 1. 회원 기능
- 회원가입
- 로그인 / 로그아웃
- 로그인 상태 유지 (세션 or 토큰)

### 2. 게시글 기능
- 전체 게시글 목록 조회 (최신순)
- 게시글 상세 보기 (글 내용 + 댓글)
- 게시글 작성 / 수정 / 삭제
- 본인 글만 수정/삭제 가능

### 3. 댓글 기능
- 댓글 등록
- 댓글 삭제 (본인만 가능)

---

## 🎯 체크리스트

- ✅ `User`, `Post`, `Comment` 엔티티 설계
- ✅ Spring Boot API 서버 구성
- ✅ REST API 명세서 작성
- ✅ DTO 계층 분리

---

## 🧪 실행 방법 (간단)

1. 백엔드: `./backend` 디렉토리에서 `./gradlew bootRun`
2. 프론트엔드: `./frontend` 디렉토리에서 `npm install` → `npm run dev`

> .env 설정이나 CORS 정책은 프론트 구성 이후 추가 예정
