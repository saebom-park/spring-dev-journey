# 🌿 spring-dev-journey: 총복습 스프린트 (10-review-sprint)

> 이 디렉토리는 지금까지 학습한 Java, JDBC, Spring, MyBatis, JPA 개념을  
> **“기억 → 복습 → 손코딩”** 흐름으로 되짚어보는 총정리 실습 시리즈입니다.

---

## ✅ 복습 철학

복습은 단순 반복이 아니라  
**"잊은 개념을 다시 떠올리고, 손으로 코드를 다시 써보며 체화하는 과정"**입니다.

- 문제를 먼저 보고 스스로 구현해보되  
- 기억이 나지 않으면 수업자료를 확인해도 됩니다  
- 핵심은 **직접 타이핑하며 다시 익숙하게 만드는 것**입니다

> 📌 기억이 나지 않아도 괜찮아요.  
> 중요한 건 **어떻게 다시 익히는가**입니다.

---

## 🧠 복습 진행 방식

1. 온이가 실습 미션(README.md)을 제시합니다.
2. 봄이는 직접 구현하며 복습을 시작합니다.
3. 중간에 개념이 막히면 👉 **수업자료를 찾아보고 다시 손코딩!**
4. 실습이 끝나면 코드 리뷰 + 실수/질문 노트를 정리합니다.
