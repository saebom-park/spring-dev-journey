# Practice01. 운영체제 개요 & 역할

## 1. 한 문장 요약

운영체제는 하드웨어와 응용프로그램 사이에서 자원을 관리하고 사용자에게
인터페이스를 제공하는 소프트웨어이다.

## 2. 5개 묶음 포인트

1.  프로세스 관리 -- 실행 관리\
2.  메모리 관리 -- 공간 할당/해제\
3.  파일 관리 -- 생성/삭제/읽기/쓰기\
4.  입출력 관리 -- 장치 접근 제어\
5.  보안 관리 -- 권한 제어 및 시스템 보호

## 3. 면접 말하기 연습 (Q&A)

-   Q: 운영체제가 무엇인가요?\
    A: 운영체제는 하드웨어와 응용프로그램 사이에서 자원을 관리하고
    사용자 인터페이스를 제공하는 소프트웨어입니다.

-   Q: 운영체제의 역할을 설명해보세요.\
    A: 운영체제의 역할은 다섯 가지입니다. 프로세스 관리, 메모리 관리,
    파일 관리, 입출력 관리, 보안 관리입니다.

-   Q: 커널과 시스템 콜의 관계를 설명해보세요.\
    A: 커널은 운영체제의 핵심이고, 시스템 콜은 응용 프로그램이 커널
    기능을 사용할 수 있도록 연결하는 인터페이스입니다.

## 4. 핵심 요약

-   운영체제 = 자원 관리 + 사용자 인터페이스 제공
-   5가지 역할 = 프로세스 / 메모리 / 파일 / 입출력 / 보안
-   구조 = 커널 / 시스템 콜 / 쉘
